All Features

Professional recording studio in your pocket

RecordIt combines an OBS-inspired studio interface with a full drawing whiteboard — all wrapped in Alvonia UI's modern Windows 11 design language.

🎛️ Studio Interface

5 Dockable Panels

Scenes, Sources, Audio Mixer, Options, and Controls panels sit below the live preview — just like a professional broadcast studio. Each panel can be shown or hidden from the View menu.

Full Menu Bar

File / Edit / View / Docks / Tools / Help menu bar gives quick access to all features. Start and stop recording directly from the File menu.

Scene Management

Create multiple scenes for different recording setups. Reorder them with arrow buttons and switch instantly via the quick-access scene bar on the preview.

Source Management

Add, remove, and reorder capture sources per scene. Each source shows its type icon (display, window, camera, audio input, audio output). Add Source dialog enumerates all live devices.

Modern Windows 11 Look

Alvonia UI's indigo accent color, rounded corners, layered dark surfaces, and Fluent Design depth make RecordIt distinctly its own — inspired by pro tools, designed for Windows 11.

Three Apps, One Design

The same studio layout and Alvonia UI tokens are shared across WinUI 3, Avalonia UI (.NET cross-platform), and Electron — identical look and feel everywhere.

🎚️ Audio Mixer

Real-Time VU Meters

Live peak-level meters per channel updating at 50 ms, powered by the Windows Core Audio IAudioMeterInformation COM interface. Green → yellow → red gradient shows safe, warning, and clip zones.

Per-Channel Volume Faders

Horizontal volume slider per channel with live dB readout. Range is 0–100% with sub-1% step precision. Volume is applied to the ffmpeg recording pipeline.

Mute Buttons

Instantly mute any channel during recording. Muted channels show flat VU meters. State is preserved between sessions.

Desktop Audio Channel

Dedicated channel for system audio (speakers / loopback). Metered via Windows default render endpoint. Recorded using WASAPI loopback capture.

Mic/Aux Channel

Dedicated microphone channel metered via the default capture endpoint. Device selectable from the Options panel from a fully enumerated device list.

Add Audio Sources

Click + in the Audio Mixer toolbar to add any detected dshow audio device as a new mixer channel with its own VU meter, fader, and mute button.

⏺ Screen Recording

4K Ultra HD Recording

Record at resolutions from 480p up to 4K Ultra HD (3840×2160) with frame rates of 15, 24, 30, or 60 fps via ffmpeg's libx264 codec.

Multiple Capture Sources

Primary display, all displays, or specific application window. Sources listed in the Sources panel with add/remove/reorder controls.

Correct Audio Mapping

Fixed: audio stream index now correctly tracks whether a webcam input is present, eliminating the silent-recording bug from earlier versions.

Webcam Overlay

Add a picture-in-picture webcam overlay at bottom-right. Select your camera from the Options panel's enumerated device list.

Desktop Audio (Loopback)

Capture system audio using WASAPI loopback — records exactly what you hear without needing virtual audio cables or Stereo Mix.

Recording Status Badge

Live red recording badge with elapsed timer overlaid on the preview. Title bar also shows the REC indicator and timer.

✏️ Whiteboard (MS Teams-Style)

Rich Drawing Tools

Pen, highlighter, eraser, and a full set of shapes: lines, rectangles, ellipses, and triangles. Adjustable brush sizes.

Text & Sticky Notes

Add text annotations anywhere on the canvas. Create colorful sticky notes for ideas, reminders, and feedback.

Color Palette

10-color quick-access palette plus a full color picker. Custom colors saved per session.

Undo / Redo

Full undo/redo history. Keyboard shortcuts (Ctrl+Z, Ctrl+Y) for fast corrections.

Participants Panel

See who is collaborating on the whiteboard in real-time. Participant cursors and presence indicators just like MS Teams.

Export to PNG

Save your whiteboard as a high-resolution PNG image. Works on both the WinUI 3 and Electron apps.

📁 Recording Library

Organized Grid View

Browse all your recordings and whiteboards in a clean thumbnail grid. Sort by date, name, or file size.

Search & Filter

Quickly find recordings by name. Filter by type (video or whiteboard) with one click.

Storage Stats

Track total storage usage, recording counts, and disk space at a glance from the library dashboard.

🎨 Alvonia UI Design System

Dark & Light Themes

First-class dark mode (default) and light mode. Toggle instantly from the title bar. Respects system preference on all platforms.

Three-App Parity

Identical studio layout, color system, typography, and interactions across WinUI 3, Avalonia UI, and Electron. Same design tokens everywhere.

Custom Title Bar

Frameless window with a custom title bar showing the app logo, live recording indicator with elapsed timer, theme toggle, and window controls.

Indigo Accent Identity

Panel headers use indigo uppercase tracking labels (VS Code / Figma-style), distinguishing RecordIt from OBS's flat gray aesthetic.

✏️ Whiteboard (MS Teams-Style)

Rich Drawing Tools

Pen, highlighter, eraser, and a full set of shapes: lines, rectangles, ellipses, and triangles. Adjustable brush sizes.

Zoom Controls

Zoom in and out (25%–400%) with smooth pan/drag. Zoom controls persist across app navigation.

Undo / Redo

Full undo/redo history. Keyboard shortcuts (Ctrl+Z, Ctrl+Y) for fast corrections.

Export to PNG

Save your whiteboard as a high-resolution PNG image from any app variant.
