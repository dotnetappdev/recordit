Features

Everything you need to record and collaborate

OBS-inspired studio interface with Alvonia UI — three apps, one design

OBS-Style Studio

5 dockable panels — Scenes, Sources, Audio Mixer, Options, and Controls — with a full menu bar and live preview. Built for professionals.

Metered Audio Mixer

Real-time VU meters per channel via Core Audio, per-channel volume faders, mute buttons, and WASAPI loopback for desktop audio.

4K Screen Recording

Capture at up to 4K at 60 fps with ffmpeg's libx264 codec. Desktop audio via WASAPI loopback — no virtual cables needed.

Teams-Style Whiteboard

Collaborative whiteboard with pen, shapes, text, sticky notes, zoom (25–400%), and real-time participant cursors — just like MS Teams.

Three-App Parity

Identical studio layout across WinUI 3, Avalonia UI (.NET cross-platform), and Electron. Same Alvonia UI design tokens everywhere.

Recording Library

Browse, preview, and manage all your recordings. Search, filter by type, and track storage usage in a clean thumbnail grid.

Three Apps, One UI

Same Alvonia UI. Every platform.
