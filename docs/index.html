Features

Everything you need to record and collaborate

Built with Alvonia UI — identical experience on Windows native and Electron

HD Screen Recording

Capture at up to 4K resolution with 60fps. Record screen, window, or specific app regions with hardware acceleration.

Teams-Style Whiteboard

Collaborative whiteboard with pen, shapes, text, sticky notes, and real-time participant cursors — just like MS Teams.

Dual App Parity

100% identical UI and features between the WinUI 3 Windows app and the Electron cross-platform app — Alvonia UI everywhere.

MSIX Installer

Clean Windows installation via MSIX package. Microsoft Store-ready with auto-updates, system tray, and startup integration.

Audio & Camera

Record with microphone audio, system sounds, and webcam overlay. Fine-grained audio mixing controls.

Recording Library

Browse, preview, and manage all your recordings. Search, filter, and organize video and whiteboard content in one place.

Two Apps, One UI

Same Alvonia UI. Different platforms.
