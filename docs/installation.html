System Requirements

  • 🖥️ OS: Windows 10 (1903+) or Windows 11
  • 💾 RAM: 4 GB minimum, 8 GB recommended
  • 🎮 GPU: DirectX 11 compatible (for hardware acceleration)
  • 💿 Storage: 200 MB for app + space for recordings
  • 🔷 Runtime: .NET 8 (for WinUI app) · Electron bundles Node.js

WinUI 3 Windows Native App (MSIX)

MSIX Package Recommended

Native Windows installer via MSIX. Clean installation, auto-updates, and Microsoft Store integration. Best performance.

⬇ Download MSIX (x64)
# Install via winget winget install Alvonia.RecordIt

Build from Source

Build the WinUI 3 app yourself using Visual Studio 2022 with the Windows App SDK workload.

git clone https://github.com/dotnetappdev/recordit cd recordit/winui # Open RecordIt.sln in Visual Studio 2022 # Build → RecordIt.Package (x64)

MSIX Installation Details

Installing the MSIX Package

  1. Download RecordIt.msix from the releases page
  2. Double-click the file to open the Windows App Installer
  3. Click "Install" (you may need to trust the certificate first)
  4. RecordIt will appear in your Start Menu and taskbar

Trusting the Certificate (Dev Builds)

For development/unsigned builds, you'll need to trust the certificate:

# PowerShell (as Administrator) $cert = Get-PfxCertificate -FilePath RecordIt.Package_TemporaryKey.pfx Import-Certificate -CertStoreLocation Cert:\LocalMachine\Root -Cert $cert
